Problem Statement

It is required to design a cotter joint to two steel rods of equal diameter. Each radius is subjected to an axial tensile force of LOAD kN design the joint and specify it's main dimensions.

Solution:

(I)    Selection of Material:
        On the basis of strength, the material of the two rods and cotter is selected as Material of grade Grade
         (Syt = stress N/mm^2)

(II)   Selection of Factor of Safty:
         Selection of factor of safty for rod spigot end and socket end is assumed as FOS while cotter is taken
         as FOS1.

(III) Calculation of permissible stresses:
         The permissible stresses rods, spigot end and socket end as follows:

          Sigma_t = Syt/fs = stress/ FOS = Sigma_t N/mm^2

          Sigma_c = 2(Syt)/fs = 2(stress)/ FOS = Sigma_C N/mm^2

          Tau = 0.5(Syt)/fs = 0.5(stress)/ FOS = Tau N/mm^2

          The permissible stresses rods, spigot end and socket end as follows:

           Sigma_t = Syt/fs = stress/ FOS1 = Sigma_t1 N/mm^2

           Tau = 0.5(Syt)/fs = 0.5(stress)/ FOS1 = Tau1 N/mm^2

(III) Calculations:

          1. Failure of the rod in tension due to the tensile load P. We know that,

               P = Pi/4 x d^2 x Sigma_t

               Load = Pi/4 x d^2 x  Sigma_t

               d = d mm

           2. Failure of Spigot in tension across the weakest section (On Slot)

               P = [Pi/4 x d2^2 - d2 x t]Sigma_t

               Load = [Pi/4 x d2^2 - d2 x t]Sigma_t                                   :.(Take t = d2/4)

                       d2 = d2 mm
               and,   t = t mm

3. Failure of rod or cotter in crushing

                P = d2 x t x Sigma_c

                Load = d2 x t x Sigma_c

                Sigma_c = Sigma_c N/mm^2

              Therefore, Sigma_c induced = Sigma_cind  N/mm^2 < Sigma_c permissible = Sigma_c N/mm^2
              Therefore, Design is Safe.

4. Failure of socket in tension across the slot.

               P = {Pi/4[(d1^2) - (d2^2)] - (d1 - d2)t}Sigma_t

               Load = {Pi/4[(d1^2) - (d2^2)] - (d1 - d2)t}Sigma_t

               d1 = d1 mm

5. Failure of cotter in shear

               P = 2 x b x t x Tau

               Load = 2 x b x t x Tau

               b = b mm

6. Failure of socket collor in crushing

               P = (d4 - d2)t x Sigma_t

               Load = (d4 - d2)t x Sigma_t1

               d4 = d4 mm

7. Failure of socket end in shearing

               P = 2(d4 - d2)c x Tau

                Load = 2(d4 - d2)c x Tau1

                c = c mm

8. Failure of rod end in shear

                P = 2a x d2 x Tau

                Load = 2a x d2 x Tau1

                a = a mm

9. Failure of spigot collor in crushing

                P = Pi/4[(d3^2) - (d2^2)]Sigma_c

                Load = Pi/4[(d3^2) - (d2^2)]Sigma_c1

                 d3 = d3 mm

10. Failure of spigot collor in shearing

                P = Pid2 x t1 x Tau

                Load = Pi x d2 x t1 x Tau

                t1 = t1 mm

11. Failure of cotter in bending

              Sigma_b = Mmax/z = ((P/2)(((d4 - d2)/6) + (d2/4)))/(t x (b^2/6)) = ((P(d4 + (0.5d2)))/(2t x b^2))

              Sigma_b = Mmax/z = ((P/2)(((d4 - d2)/6) + (d2/4)))/(t x (b^2/6)) = ((P(d4 + (0.5d2)))/(2t x b^2))

              Sigma_b = Sigma_b N/mm^2

12. The length of cotter l is taken as 4d

               l = 4d

               l = 4 x d

               l = l mm

13. The tapper in cotter should be not be exceed 1 in 24. In case the grater tapper is required then a locking device must be

              tan theta = 1/24

14. The draw of cotter is generally taken as 2 to 3 mm
